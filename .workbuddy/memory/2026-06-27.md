## JWT 解码加入签名验证

### 改动内容
JwtPanel.java 解码面板新增功能：
- 添加**签名密钥输入框**（JPasswordField），支持显示/隐藏切换
- 解码时自动验证签名（支持 HS256 / HS384 / HS512）
- 从 Header 中解析 `alg` 字段确定算法，重新计算 HMAC 并与签名段对比
- 状态栏显示 ✅ 签名验证通过 / ❌ 签名验证失败
- 演示 JWT 已预填密钥 "your-256-bit-secret"，打开即可看到验证结果
