## JWT 解码加入签名验证

### 改动内容
JwtPanel.java 解码面板新增功能：
- 添加**签名密钥输入框**（JPasswordField），支持显示/隐藏切换
- 解码时自动验证签名（支持 HS256 / HS384 / HS512）
- 从 Header 中解析 `alg` 字段确定算法，重新计算 HMAC 并与签名段对比
- 状态栏显示 ✅ 签名验证通过 / ❌ 签名验证失败
- 演示 JWT 已预填密钥 "your-256-bit-secret"，打开即可看到验证结果

## 证书管理工具

### 改动内容
新增证书管理功能（开发工具分组），包含 3 个标签页：
1. **创建根证书** - 选择密钥算法(RSA 2048/4096, EC P-256/384/521)，填写主题 DN，生成自签名根 CA 证书和私钥 PEM
2. **签发证书** - 导入 CA 证书和私钥，指定新证书主题和 SAN 域名，由 CA 签发终端证书
3. **证书解析** - 粘贴 PEM 证书，解析显示 Subject/Issuer/Serial/有效期/公钥/指纹等详细信息

### 新增文件
- `crypto/CertUtils.java` - 证书工具类（创建、签发、解析、PEM 编解码）
- `misc/CertPanel.java` - UI 面板（3 个标签页）
