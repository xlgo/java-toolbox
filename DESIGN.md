# Design System

## Product register

面向开发者和运维人员的 Java Swing 桌面产品界面。设计服务于高频任务，强调专业、克制和高效。

## Layout

- 主窗口由一个分组侧边导航、一个内容工作区和底部状态栏组成。
- 侧边栏默认宽度 228px，最小 190px，最大 320px。
- 侧边栏支持收起，并持久化宽度、收起状态、分组展开状态和当前工具。
- 当前工具栏始终位于内容区上方；状态信息始终位于窗口底部。
- 具体工具继续使用自己的布局，不在外层增加统一滚动容器。

## Spacing and density

- 间距刻度：4px、8px、12px、16px。
- 导航行高：32px。
- 常规按钮、文本框和下拉框：至少 32px 高。
- 工具内容继续使用 `UIUtils.CONTENT_PADDING`，除非该工具单独重构。

## Color and theme

- 颜色来自 FlatLaf 和 `UIManager`，不硬编码浅色或深色壳层颜色。
- 选中、焦点、禁用、错误、警告和成功状态遵循当前主题。
- 使用一条细分隔线区分当前工具栏和内容，不增加装饰性卡片或阴影。

## Typography

- 使用现有平台字体和 FlatLaf 标签字体。
- 应用名称、分类和当前工具通过字重建立层级。
- 水平空间不足时截断工具名称，并用 tooltip 提供完整文本。

## Interaction

- `Ctrl+K` 聚焦搜索；侧边栏收起时会先展开。
- 方向键移动工具选择，选中项与右侧内容保持同步。
- `Enter` 打开当前工具或展开分类，`Esc` 清空搜索。
- 搜索只过滤导航，不销毁已经打开的工具视图。
- 分类单击展开或收起，侧边栏可拖动调整宽度。

## Accessibility

- 键盘焦点必须可见，选中状态不能只靠颜色表达。
- 搜索和导航提供本地化的可访问名称。
- 简体中文与 English 切换不会改变工具稳定 ID 或清空当前工具状态。
- 820×520 最小窗口下，全局导航、主题、语言和内容仍可到达。
