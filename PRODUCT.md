# Java 工具箱

## Register

product

## Users

面向需要在桌面环境中高频处理开发、测试、运维和数据转换任务的开发者与运维人员。用户通常在普通办公光线下使用 24 至 27 英寸显示器，也可能把窗口缩小后与 IDE、终端或浏览器并排操作。

## Product Purpose

将加密、转换、格式化、接口调试、数据管理、运维和远程协作等工具集中到一个本地 Java Swing 客户端中。成功标准是用户能快速定位工具、理解当前上下文、完成操作，并在不同窗口尺寸和主题下保持稳定、清晰的布局；现有工具与能力不得减少。

## Brand Personality

专业、克制、高效。界面服务于任务，不用装饰抢占注意力；信息密度适合高频开发工具，同时避免拥挤和层级混乱。

## Anti-references

不采用网页落地页式的大卡片、巨幅留白、装饰性动画或为了视觉效果而改变标准桌面控件行为。不把导航拆成重复层级，不用固定尺寸造成小窗口裁切，也不通过隐藏功能来换取表面整洁。

## Design Principles

1. 统一导航层级，让工具定位路径短且可预测。
2. 紧凑但不拥挤，优先展示当前任务和主要操作。
3. 使用熟悉的桌面交互模式，保持键盘、鼠标和搜索操作一致。
4. 通过共享布局规范解决跨页面问题，不做互不一致的局部美化。
5. 保留全部功能、主题和语言能力，并让窗口缩放时核心操作始终可达。

## Accessibility & Inclusion

保留中英文切换和 FlatLaf 全部主题，遵循当前主题的对比度与焦点状态。支持键盘遍历、清晰的选中与焦点反馈，避免仅依赖颜色表达状态；在应用最小窗口尺寸下不得遮挡或丢失核心操作。
