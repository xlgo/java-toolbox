# 微信导出与通讯录工具 (WeChat Tools) 使用指南

## 概述

工具箱包含两套互补的微信辅助工具：
1. **微信群发与通讯录解析 (`WeChatPanel` & `WeChatContactReader`)**：纯 Java 实现，直接解析本地 SQLite 数据库。
2. **微信 UI 自动化导出脚本 (`tools/wechat_export.py`)**：Python 实现，基于 Windows UIAutomation 库，模拟 UI 点击进行消息导出，并配备独立可视化悬浮控制面板。

---

## 模块一：本地通讯录读取 (`WeChatContactReader`)

### 1. 原理
微信桌面端将通讯录与聊天记录存储于本地 `MicroMsg.db` / `MSG*.db` 的 SQLite 数据库中。本模块自适应扫描默认安装目录与用户数据路径，解密/读取联系人列表。

### 2. 功能特性
- **基本信息解析**：提取好友与群聊的微信昵称、备注名、微信号、头像 URL 及分组。
- **数据导出**：支持一键导出包含完整的联系人 Excel 表格。
- **头像批量下载**：自动异步下载选定联系人或全量好友的高清头像至本地目录。
- **群发联动**：支持勾选多个联系人后一键追加至群发消息列表。

---

## 模块二：UI 自动化导出脚本 (`tools/wechat_export.py`)

### 1. 简介
由于微信新版数据库加密机制频繁升级，`tools/wechat_export.py` 提供了一种无需依赖数据库密钥的无痕 UI 自动化导出方案。工具提供 Tkinter 悬浮窗控制面板，便于在导出过程中随时暂停或终止。

```
 ┌────────────────────────────────────────┐
 │   微信自动化导出控制台                │
 ├────────────────────────────────────────┤
 │ 已导出会话数: 42                       │
 │ [ ⏸ 暂停 / 继续 ]    [ ⏹ 停止导出 ]     │
 └────────────────────────────────────────┘
```

### 2. 核心特性
- **浮动控制面板 (Floating Control Window)**：悬浮在屏幕最上层，避免自动点击过程中界面失控。支持随时 `暂停/继续` 和 `安全终止`。
- **坐标防失焦校验**：自动校验元素滚动后的物理 `BoundingRectangle`。如果列表项滚出可见区域或虚拟化失效，自动触发 `ScrollIntoView` 并校验有效性，防止无效 Click 崩溃。
- **自动化流控制**：按设定的延迟时间逐个遍历会话框并自动触发消息历史导出。

### 3. 环境准备与运行
环境要求：Windows 系统，Python 3.x（脚本会自动尝试通过 `pip` 动态补全依赖 `uiautomation`）。

```bash
# 启动导出脚本
python tools/wechat_export.py --delay 0.5
```
